## Промокод на билеты

Для покупки билетов можно использовать промокод:

```text
DX749988
```

Промокод дает **скидку 10% / 100% на сервисный сбор** при покупке билетов от 3 000 ₽ на Яндекс Афише.

Промокод действует **до 1 августа 2026 года** на это мероприятие.

### Как применить промокод

1. Перейдите на страницу покупки билетов.
2. Выберите билет на концерт Юлии Савичевой.
3. На странице подтверждения заказа нажмите **«Ввести промокод»**.
4. Введите промокод `DX749988`.
5. Проверьте, что скидка применилась до оплаты.
6. После этого нажмите **«Оплатить билет»**.

Промокод нужно применить **до оплаты заказа**. После покупки билета добавить скидку к уже оплаченному заказу нельзя.
